Limpieza y Preparación de Datos

Author

Daniel Arcos

Propósito de esta sección

En este documento se describe cómo se prepararon los datos antes del análisis visual. Dado que el archivo contiene millones de registros, se trabajó con un subconjunto de variables relevantes para las preguntas del proyecto, manteniendo el foco en tiempo, demanda, distancia, costo y duración del viaje.

La estrategia de limpieza es conservar únicamente registros completos en las variables esenciales para el análisis. Esto evita que faltantes en distancia, costo o fechas afecten la lectura de los resultados.

essential_cols = [
    "pickup_datetime",
    "dropoff_datetime",
    "passenger_count",
    "trip_distance",
    "total_amount",
]

df_clean = df.dropna(subset=essential_cols).copy()

Transformaciones y creación de variables

Se crean variables derivadas para responder mejor las preguntas analíticas:

  • trip_duration_min: duración del viaje en minutos
  • pickup_hour: hora del día del viaje
  • pickup_day_name: día de la semana
df_clean["trip_duration_min"] = (
    (df_clean["dropoff_datetime"] - df_clean["pickup_datetime"])
    .dt.total_seconds()
    / 60
)

df_clean["pickup_hour"] = df_clean["pickup_datetime"].dt.hour

df_clean["pickup_day_name"] = pd.Categorical(
    df_clean["pickup_datetime"].dt.day_name(),
    categories=[
        "Monday", "Tuesday", "Wednesday", "Thursday",
        "Friday", "Saturday", "Sunday"
    ],
    ordered=True
)

Filtrado o selección de datos

La limpieza se enfoca en eliminar registros poco plausibles o claramente problemáticos para el análisis:

  1. Se eliminan viajes con passenger_count <= 0.
  2. Se eliminan viajes con trip_distance <= 0.
  3. Se eliminan viajes con total_amount <= 0.
  4. Se eliminan viajes con duración menor a 1 minuto o mayor a 180 minutos.
df_clean = df_clean[df_clean["passenger_count"] > 0]
df_clean = df_clean[df_clean["trip_distance"] > 0]
df_clean = df_clean[df_clean["total_amount"] > 0]
df_clean = df_clean[df_clean["trip_duration_min"].between(1, 180)]

La base limpia se guarda como data/df_clean.pkl para que el documento de visualizaciones reutilice exactamente el mismo df_clean y no repita el proceso de limpieza.

Justificación de las decisiones tomadas

Las decisiones de preparación se tomaron para asegurar consistencia analítica:

  • Se seleccionaron solo las columnas necesarias para reducir carga computacional y concentrar el análisis.
  • Se eliminaron valores faltantes en variables esenciales porque impiden calcular o interpretar adecuadamente los indicadores.
  • Se filtraron distancias, costos y duraciones no positivas porque no representan viajes válidos.
  • Se acotó la duración entre 1 y 180 minutos para reducir el efecto de registros extremos o errores de captura.

La tabla df_clean es la base utilizada en el documento de visualizaciones.
